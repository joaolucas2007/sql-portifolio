## Sobre o Projeto

## 🎯 Objetivo do Repositório
A SeriesDB simula um sistema de gerenciamento de séries de streaming, permitindo o cadastro e a análise de informações relacionadas às séries e suas plataformas.

Este repositório centraliza a aplicação prática de conceitos fundamentais de engenharia e modelagem de dados utilizando o **SQL Server**. O foco principal é demonstrar a resolução de problemas de negócio reais através de código limpo, estruturado e performático.
O projeto foi criado com o objetivo de consolidar conhecimentos em modelagem de dados, relacionamentos entre tabelas e desenvolvimento de consultas SQL.

### Principais Competências Aplicadas:
* **Modelagem Relacional:** Arquitetura de dados focada em integridade referencial.
* **Desenvolvimento de Schemas:** Criação de tabelas, definição de tipos de dados e constraints de integridade (PK/FK).
* **Operações de CRUD:** Manipulação segura de dados, cargas iniciais (Inserts), atualizações orientadas a ID (Updates) e deleções controladas (Deletes).
* **Inteligência de Negócio (Relatórios):** Extração de dados complexos através de Joins, funções de agregação (`Sum`), agrupamentos (`Group By`), formatação de dados (`Convert`) e filtros lógicos avançados (`Like`/`Or`).
* **Boas Práticas de Engenharia:** Padronização rigorosa de escrita de código através da convenção PascalCase.
